# Files in `.dna/`

Everything dna writes lives under `.dna/` in your repo. Commit what you want to share with your team and agents.

```
.dna/
├── config.yml              # languages, exclusions, depth, strands       ✅ commit
├── invariants.yml          # author-defined rules                        ✅ commit
├── index.json              # symbol graph (regenerated)                  ❌ gitignore
├── preferences.json        # captured user preferences                   ✅ commit
├── observations.json       # per-symbol query counts (opt-in)            ❌ gitignore
├── features.json           # feature → symbols attribution               ✅ commit
├── notes/
│   ├── symbol/             # `.dna/notes/symbol/<symbol-id>.json`         ✅ commit
│   ├── file/               # `.dna/notes/file/<path>.json`               ✅ commit
│   └── feature/            # `.dna/notes/feature/<label>.json`           ✅ commit
├── decisions/              # `.dna/decisions/<symbol-id>.json`           ✅ commit
└── lessons/                # raw lessons before classification           ✅ commit
```

## config.yml

```yaml
languages: [typescript, python]
exclude: [node_modules, dist, build, .next, vendor, __pycache__, .venv]
depth: 3
strands:
  - structural
  - tests
  - provenance
  - invariants
```

JSON Schema: [`packages/schemas/json/config.schema.json`](../../packages/schemas/json/config.schema.json).

## invariants.yml

```yaml
- name: High-value refunds require approval
  applies_to:
    - createRefund
    - "stripe.refunds.create"
  rule: Refunds over 1000 require finance_approval_id.
  evidence:
    - docs/refund-policy.md
  severity: block      # info | warn | block
```

`applies_to` accepts symbol names or qualified-name globs. JSON Schema: [`packages/schemas/json/invariants.schema.json`](../../packages/schemas/json/invariants.schema.json).

## notes/, decisions/, lessons/

JSON files, one per symbol/file/feature. Don't hand-edit unless you have to — use `dna learn`, `dna decide`, `dna lessons record` instead. Schemas in [`packages/schemas/json/`](../../packages/schemas/json/).

## index.json

The compiled symbol graph. Regenerated by `dna index`. Add to `.gitignore` — it's a build artifact.

## observations.json

Per-symbol query counts captured by `dna serve --observe`. **Metadata only**: symbol name + timestamp. No tool arguments, no tool results, no conversation content. Add to `.gitignore` if you don't want to share local agent activity.

## Suggested `.gitignore` entries

```
.dna/index.json
.dna/observations.json
```

Everything else is durable team knowledge — commit it.
